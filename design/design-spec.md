## 三、公共组件清单

### 1. 状态栏 (StatusBar)
- 高度：44px
- 时间在左，系统图标在右
- 背景色跟随页面（白/黑）

### 2. 导航栏 (NavBar)
- 高度：44px
- 左侧：首页图标 or 返回箭头 + 标题
- 右侧：⋯ 菜单按钮 + ⚪ 功能按钮（32×32px 圆角8px）

### 3. 底部Tab栏 (TabBar)
- 高度：83px（含安全区34px）
- 5个等宽选项：首页、上新、分类、购物袋、我的
- 选中态：黑色文字 + 图标填充
- 未选中：灰色文字 + 图标线性
- 顶部细线：0.5px #eee

### 4. 商品卡片 - 列表式 (ProductCardList)
- 左图右文，圆角12px
- 左图：120×150px
- 标签、库存、价格、加号按钮

### 5. 商品卡片 - 网格式 (ProductCardGrid)
- 两列，圆角10px
- 商品图全宽160px高
- 品牌、名称、标签、库存、价格+购物车

### 6. 黑色促销横幅 (PromoBanner)
- 背景：#000，圆角0（全宽）
- 白色文字居中

### 7. 分类Tab栏 (CategoryTabs)
- 可横滑
- 选中：黑色加粗 + 下划线
- 未选中：灰色常规

## 四、开发注意事项

### 小程序适配要点

1. **rpx 转换**：设计稿 375px = 750rpx，所有尺寸需 ×2
2. **安全区**：底部Tab栏需适配 `env(safe-area-inset-bottom)`
3. **图片懒加载**：商品列表和网格图片使用 `lazy-load`
4. **下拉刷新**：首页、破价专区、上新支持下拉刷新
5. **上拉加载**：商品列表支持分页加载

### 页面路由结构

```
pages/
├── home/index          ← 首页
├── special-offer/index ← 破价专区
├── new-arrivals/index  ← 上新
├── category/index      ← 分类
├── cart/index          ← 购物袋
├── my/index            ← 我的
├── product/detail      ← 商品详情
├── search/index        ← 搜索
└── member/index        ← 会员中心
```

### TabBar 配置

| Tab | 选中图标 | 未选中图标 | 页面 |
|-----|---------|-----------|------|
| 首页 | 填充房屋 | 线性房屋 | pages/home/index |
| 上新 | 填充图层 | 线性图层 | pages/new-arrivals/index |
| 分类 | 填充网格 | 线性网格 | pages/category/index |
| 购物袋 | 填充袋子 | 线性袋子 | pages/cart/index |
| 我的 | 填充人物 | 线性人物 | pages/my/index |
